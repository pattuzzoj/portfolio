Technologies and Tools

Astro

Modern architecture for static sites. Zero JS by default and interactivity only where it matters.

Next.js

Complete React framework with SSR, SSG, and built-in routing.

Solid.js

True reactivity without a virtual DOM. Compiles to pure JS with high performance.

React

Component-based UI library. Popular, but with indirect reactivity.

Tailwind CSS

Utility-first CSS for fast and consistent UI building.

NestJS

Modular framework with TypeScript. Solid and scalable architecture by default.

Express

Minimalist framework for Node.js. Flexible, but unopinionated.

Deno

Modern runtime with built-in security, ES Modules, and native tools.

Node.js

Asynchronous runtime built on V8. Foundation of modern JS servers.

Drizzle ORM

Lightweight ORM with strong typing and zero overhead. Ideal for full control.

Prisma

Modern ORM focused on productivity. Safe, auto-generated queries.

PostgreSQL

Robust and reliable relational database. Supports JSON, ACID, and extensibility.

MongoDB

NoSQL document-oriented database. Flexible, but less strict than SQL.

Vitest

Fast test runner with ESM, HMR, and Vite integration.

Jest

Established framework with snapshot and coverage support.

Git

Distributed version control. Essential for collaboration and history tracking.

Docker

Containers for isolated environments. Standardizes dev and production.
