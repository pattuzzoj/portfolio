Sobre Mim

Desenvolvedor apaixonado por criar soluções web inovadoras e funcionais

Sou desenvolvedor Full Stack, formado em Análise e Desenvolvimento de Sistemas. Tenho uma abordagem prática e direta no desenvolvimento de software, com foco em escrever código limpo, modular e fácil de manter.

Estou em constante evolução, buscando aprimoramento técnico por meio do estudo contínuo de novas tecnologias, ferramentas e padrões de arquitetura.

Aprender faz parte do processo — evoluir é parte do trabalho.

Front-End

Desenvolvimento de interfaces modernas e responsivas com Solid.js, Astro, React e Next.js, focando em desempenho e experiência do usuário.

Back-End

Construção de APIs eficientes e escaláveis utilizando Node.js, NestJS e Express, com foco em arquitetura modular e alta performance.

Banco de Dados

Gerenciamento e manipulação de dados em PostgreSQL e MongoDB, com consultas claras e seguras via Drizzle ORM e Prisma.

Testes

Implementação de testes unitários usando Vitest e Jest para garantir qualidade e estabilidade.

Tecnologias e Ferramentas

Astro

Arquitetura moderna para sites estáticos. Zero JS por padrão e interatividade apenas onde importa.

Next.js

Framework React completo com SSR, SSG e roteamento integrado.

Solid.js

Reatividade real sem virtual DOM. Compila para JS puro com alta performance.

React

Biblioteca de UI baseada em componentes. Popular, mas com reatividade indireta.

Tailwind CSS

CSS utilitário para construir interfaces rápidas e consistentes.

NestJS

Framework modular com TypeScript. Arquitetura sólida e escalável por padrão.

Express

Framework minimalista para Node.js. Flexível, mas sem estrutura opinativa.

Deno

Runtime moderno com segurança embutida, ES Modules e ferramentas nativas.

Node.js

Runtime assíncrono baseado no V8. Base para servidores JS modernos.

Drizzle ORM

ORM leve com tipagem forte e zero overhead. Ideal para controle total.

Prisma

ORM moderno com foco em produtividade. Consultas seguras e geradas automaticamente.

PostgreSQL

Banco relacional robusto e confiável. Suporte a JSON, ACID e extensibilidade.

MongoDB

Banco NoSQL orientado a documentos. Flexível, mas menos rígido que SQL.

Vitest

Test runner rápido com ESM, HMR e integração ao Vite.

Jest

Framework consolidado com suporte a snapshots e coverage.

Git

Controle de versão distribuído. Essencial para colaboração e histórico.

Docker

Contêineres para ambientes isolados. Padroniza dev e produção.
